# 介绍

使用TypeScript语法进行组件类型定义，有效提升组件的类型安全.

## 🕸️ 组件目录

```js
📁 types/
  └── 📁src/
    └── 📄button.ts     // 按钮
    └── 📄config.ts     // 应用配置
    └── 📄global.ts     // 全局
    └── 📄menu.ts       // 菜单
    └── 📄router.ts     // 路由
    └── 📄service.ts    // 请求
    └── 📄setting.ts    // 模块设置
    └── 📄store.ts      // 状态管理
    └── 📄index.ts
```

## 🌈 快速使用

```ts
import { modulesType } from '@vben/types'
```
